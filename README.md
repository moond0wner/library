# Библиотека

Консольное приложение на C++ для управления библиотекой. Позволяет вести учёт книг и читателей, выдавать и возвращать книги, отслеживать должников.

---

## Функционал

### Управление книгами
- Добавление книги (название, автор, год)
- Удаление книги (только доступные)
- Поиск по названию и автору
- Просмотр всех, доступных и выданных книг

### Управление читателями
- Добавление читателя (имя, контакт)
- Удаление читателя (автоматический возврат всех книг)
- Поиск по имени
- Просмотр всех читателей

### Операции
- Выдача книги (проверка доступности и лимита)
- Возврат книги
- Список должников (книги, просроченные на 14+ дней)

---

## Технические детали

- **Классы**: `Book`, `Reader`, `Library`
- **Хранение данных**: `std::vector`
- **Даты**: `time_t`, функции для работы с датами
- **Валидация**: проверка доступности книги и лимита выдачи
- **Обработка ошибок**: защита от некорректного ввода
