# Библиотека

Консольное приложение на C++ для управления библиотекой.  
Позволяет вести учёт книг и читателей, выдавать и возвращать книги, отслеживать должников.  
Все данные сохраняются в JSON-файл и загружаются при следующем запуске.

---

## Функционал

### 📚 Управление книгами
- Добавление книги (название, автор, год)
- Удаление книги (только доступные)
- Выдача и возврат книги
- Поиск по названию и автору (без учёта регистра)
- Просмотр всех, доступных и выданных книг

### 👥 Управление читателями
- Добавление читателя (имя + телефон или email с валидацией)
- Удаление читателя (автоматический возврат всех книг)
- Поиск по имени (без учёта регистра)
- Просмотр всех читателей

### 🧾 Операции
- Выдача книги (проверка доступности и лимита)
- Возврат книги
- Список должников (книги, просроченные на 14+ дней)

### 🧹 Управление данными
- Очистка всех книг и читателей
- Автоматическое сохранение и загрузка из JSON

---

## Технические детали

- **Язык:** C++17
- **Сборка:** CMake
- **Хранение:** `std::vector` (в памяти) + JSON (nlohmann/json) — сохранение между запусками
- **Классы:** `Book`, `Reader`, `Library`
- **Даты:** `time_t`, формат `YYYY-MM-DD HH:MM:SS`
- **Кодировка:** UTF-8 (консоль: 65001)

## Сборка и запуск

### Windows
```bash
mkdir build && cd build
cmake ..
cmake --build .
cd Debug
./library.exe
```

### Linux / macOS
```bash
mkdir build && cd build
cmake ..
make
./library
```

## Требования

- CMake 3.10 или выше

- Компилятор C++17 (GCC 7+, Clang 5+, MSVC 2017+)

- Терминал с поддержкой UTF-8 (для корректного отображения русского текста)

- Операционная система: Windows, Linux или macOS
